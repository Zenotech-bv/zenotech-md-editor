# Zenotech MD Editor

Markdown editor with WYSIWYG, Zenotech MD Editor-branded build of [Zenotech MD Viewer](https://github.com/Zenotech-bv/zenotech-md-viewer-source).

## Download

Get the latest installer from the [**Releases**](https://github.com/Zenotech-bv/zenotech-md-editor/releases/latest) page.

Installs per-user (no admin), updates automatically within the **editor track** (`1.x.y`).

## Two product lines

| Product | Versioning | Releases |
|---|---|---|
| MD Viewer | `v0.x.y` | [zenotech-md-viewer](https://github.com/Zenotech-bv/zenotech-md-viewer)/releases |
| MD Editor | `v1.x.y` and up | this repo |

The Editor is a strict superset of the Viewer: same rendering, file handling, and security model, with WYSIWYG editing added. The two install side by side under different Windows identifiers and don't collide.

## Source

Source lives in the private [Zenotech-bv/zenotech-md-viewer-source](https://github.com/Zenotech-bv/zenotech-md-viewer-source) repo. This repository is a release-only mirror.
